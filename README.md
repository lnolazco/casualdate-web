# casualdate-web

application for casual dates :)
- LoopBack (strongloop)
- AngularJs
- MongoDB

# How to run the application

```
$ git clone https://github.com/lnolazco/casualdate-web.git
$ cd loopback-example-passport
$ npm install
$ bower install
```

## Run the application

```
$ node .
```

Open your browser to http://127.0.0.1:3000
